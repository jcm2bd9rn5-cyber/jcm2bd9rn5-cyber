# Shogo Taguchi / 田口将伍

東京電機大学 情報システム工学科で学びながら、iOS / Flutter / SwiftUI を中心に個人開発を行っています。  
アプリの企画、設計、実装、公開、改善までを一貫して経験し、App Store 公開済みアプリの開発・運用にも取り組んでいます。

ユーザーが継続して使える体験設計と、実際に動くプロダクトとして届ける実装力を重視しています。

## About

- App Store 公開済みアプリを個人開発・運用
- Flutter / Firebase を用いたモバイルアプリ開発
- SwiftUI / AVFoundation を用いた iOS ネイティブ開発
- RevenueCat を用いたサブスクリプション機能の実装
- 企画、UI設計、実装、公開後改善まで一貫して担当
- ITコンサル / プロダクト開発領域に関心

## Featured Projects

### Strive

筋トレ記録と継続支援に特化した iOS アプリです。  
前回重量・回数の表示、PR管理、インターバルタイマー、履歴、フレンド機能、ランキング、月間レポート、サブスクリプション機能などを実装しています。

**Tech:** Flutter / Dart / Firebase Auth / Cloud Firestore / RevenueCat / App Store Connect  
**Status:** Published on the App Store

### Nolune

Dynamic Island 風のカメラ体験と、チェキ風に思い出を残せるアルバムアプリです。  
SwiftUI と AVFoundation を用いて、撮影体験、アルバム UI、アニメーション表現を設計・実装しています。

**Tech:** SwiftUI / AVFoundation / iOS  
**Status:** In development

### Focas

ポモドーロと集中時間の記録を組み合わせた集中支援アプリです。  
学習・作業時間を可視化し、シンプルに継続できる操作体験を重視して開発しています。

**Tech:** React Native / Expo / TypeScript  
**Status:** In development

### StudyMatch

学習目標を持つ学生同士が学習パートナーを見つけるための Web アプリです。  
目標、使用技術、学習時間帯などをもとに、相性の良い学習相手を探せる体験を設計しました。

**Tech:** Next.js / TypeScript / Tailwind CSS / Prisma / PostgreSQL  
**Status:** Portfolio project
